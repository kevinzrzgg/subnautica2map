# Subnautica 2 Map

Unofficial fan-made Subnautica 2 interactive map and guide site for blueprints, fragments, materials, recipes, vehicles, and biomes.

## Local development

```bash
npm install
npm run dev
```

## Static build for Cloudflare Pages

```bash
npm run build
```

Cloudflare Pages settings:

- Framework preset: Next.js
- Build command: `npm run build`
- Build output directory: `out`
- Production branch: `main`

## Custom domain

Primary domain: `https://subnautica2map.click`
